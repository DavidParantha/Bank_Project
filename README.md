# Bank_Project

AceBank Lite is a Jakarta Servlet/JSP banking web application packaged as a Maven WAR.

## Local Run

Use Tomcat 10.1+ or Tomcat 11 with JDK 21.

```powershell
.\run-local-tomcat.ps1
```

Then open:

```text
http://localhost:8080/
```
